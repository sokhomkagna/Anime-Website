# 🎌 OTAKU Anime Website

## 📖 Description
OTAKU is a responsive anime website that showcases trending, popular, and latest anime releases.  
It features an interactive UI with sliders, search functionality, and a sidebar navigation menu.

This project is built using HTML5, CSS, Tailwind CSS, and JavaScript.

---

## 🚀 Features

- 🎥 Hero slider (Swiper.js)
- 📱 Fully responsive design (mobile, tablet, desktop)
- 📂 Sidebar navigation menu (hamburger menu)
- 🔍 Search bar toggle
- 📺 Trending anime slider
- 🆕 Latest release section
- 🎨 Clean and modern UI design

---

## 🛠️ Technologies Used

- HTML5 (Semantic structure)
- CSS3
- Tailwind CSS
- JavaScript (Vanilla JS)
- Swiper.js (for sliders)
- Font Awesome (icons)
